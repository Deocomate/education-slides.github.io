Session 3: CSS Box Model & Layout

Nền tảng để xây dựng mọi giao diện Web

Hãy cùng tìm hiểu cách các phần tử chiếm không gian trên trang web!

1. CSS Box Model là gì?

Trong CSS, mọi phần tử HTML đều có thể được coi là một chiếc hộp (box). Box Model là một khái niệm mô tả cách các hộp này được hiển thị và chiếm không gian trên trang.

Mỗi hộp bao gồm 4 vùng, từ ngoài vào trong:

Margin (Vùng lề ngoài)
Border (Đường viền)
Padding (Vùng đệm trong)
Content (Nội dung)
  • Content: Vùng chứa nội dung thực tế (văn bản, hình ảnh, ...).
  • Padding: Vùng đệm trong suốt xung quanh nội dung, nằm bên trong đường viền.
  • Border: Một đường viền bao quanh padding và content.
  • Margin: Vùng lề trong suốt nằm bên ngoài đường viền, tạo khoảng cách với các phần tử khác.

2. Margin - Lề Ngoài

margin dùng để tạo khoảng trống bên ngoài đường viền của một phần tử, đẩy các phần tử khác ra xa.

<style>
.box1 {
    background-color: #fecaca;
    border: 2px solid #b91c1c;
    width: 200px;
    padding: 10px;
    /* Đẩy box thứ 2 cách xa 50px về phía dưới */
    margin-bottom: 50px;
}
.box2 {
    background-color: #dbeafe;
    border: 2px solid #1d4ed8;
    width: 200px;
    padding: 10px;
    /* Tự động căn giữa box theo chiều ngang */
    margin-left: auto;
    margin-right: auto;
}
</style>
<div class="box1">Box 1</div>
<div class="box2">Box 2 (được căn giữa)</div>

3. Padding - Đệm Trong

padding dùng để tạo khoảng trống bên trong đường viền, giữa viền và nội dung.

<style>
.button-no-padding {
    background-color: #6ee7b7;
    border: 2px solid #047857;
}
.button-with-padding {
    background-color: #6ee7b7;
    border: 2px solid #047857;
    /* Thêm khoảng đệm 15px xung quanh chữ */
    padding: 15px;
}
</style>
<button class="button-no-padding">Nút không Padding</button>
<br><br>
<button class="button-with-padding">Nút có Padding</button>

4. Cách viết tắt cho Margin & Padding

Bạn có thể thiết lập giá trị cho cả 4 phía cùng lúc để code ngắn gọn hơn.

  • 4 giá trị: margin: 10px 20px 30px 40px; (trên, phải, dưới, trái)
  • 3 giá trị: margin: 10px 20px 30px; (trên, trái/phải, dưới)
  • 2 giá trị: margin: 10px 20px; (trên/dưới, trái/phải)
  • 1 giá trị: margin: 10px; (tất cả 4 phía)

Quy tắc tương tự áp dụng cho padding.

<style>
.shorthand-box {
    background-color: #f5d0fe;
    border: 2px solid #86198f;
    /* trên/dưới: 20px, trái/phải: 40px */
    padding: 20px 40px;
}
</style>
<div class="shorthand-box">Box này dùng padding shorthand.</div>

5. Đường viền: Border

Thuộc tính border cho phép bạn chỉ định kiểu, độ dày và màu sắc của đường viền.

Cách viết tắt (shorthand):

border: [độ dày] [kiểu] [màu];

  • border-style: Kiểu viền (solid, dashed, dotted, double, groove, ...).
  • border-radius: Dùng để bo tròn các góc của phần tử.
<style>
  .box-border { padding: 20px; margin-bottom: 15px; background-color: #f1f5f9; }
  .solid-border { border: 2px solid #2563eb; }
  .dashed-border { border: 3px dashed #dc2626; }
  .rounded-border { border: 2px solid #16a34a; border-radius: 15px; }
</style>
<div class="box-border solid-border">Viền liền (solid).</div>
<div class="box-border dashed-border">Viền đứt nét (dashed).</div>
<div class="box-border rounded-border">Viền được bo tròn (border-radius).</div>

Kết quả:

6. Kích thước & Box-sizing

width và height thiết lập kích thước cho vùng content.

Điều này có thể gây ra vấn đề: width: 100px + padding: 20px => chiều rộng thực tế là 140px!

Giải pháp: `box-sizing: border-box;`

Khi dùng thuộc tính này, padding và border sẽ được tính bên trong width và height đã cho, chứ không cộng thêm vào. 

<style>
  .box-sizing {
    width: 200px;
    padding: 20px;
    border: 10px solid #fb923c;
    margin-bottom: 10px;
    color: white;
  }
  .content-box {
    background-color: #1e40af;
    /* Chiều rộng thực: 200 + 20*2 + 10*2 = 260px */
    box-sizing: content-box; /* Mặc định */
  }
  .border-box {
    background-color: #b91c1c;
    /* Chiều rộng thực: 200px */
    box-sizing: border-box;
  }
</style>
<div class="box-sizing content-box">content-box (260px)</div>
<div class="box-sizing border-box">border-box (200px)</div>

Kết quả:

7. `max-width` cho Responsive Design

max-width là một thuộc tính cực kỳ quan trọng để xây dựng layout đáp ứng (responsive).

  • Nó đảm bảo phần tử không bao giờ rộng hơn giá trị đã cho.
  • Nhưng nó sẽ tự động co lại trên các màn hình nhỏ hơn.
  • Kết hợp max-width với margin: auto là cách phổ biến để tạo và căn giữa các container chính của trang web.
<style>
.container {
    max-width: 600px; /* Rộng tối đa 600px */
    margin: 20px auto; /* Căn giữa container */
    padding: 20px;
    background-color: #fff;
    border: 1px solid #ddd;
    border-radius: 8px;
}
</style>
<div class="container">
  <h3>Container Responsive</h3>
  <p>Hãy thử thay đổi kích thước cửa sổ trình duyệt. Nội dung này sẽ không bao giờ rộng hơn 600px, nhưng nó sẽ co lại nếu cửa sổ nhỏ hơn.</p>
</div>

8. Bài tập: Nâng cấp trang Profile

Bây giờ, chúng ta sẽ áp dụng các kiến thức vừa học để "lột xác" cho trang Profile đã tạo ở buổi 1.

HTML gốc từ Buổi 1:

<!-- Chúng ta sẽ bắt đầu với đoạn code này -->
<h1>Hello, I'm John Doe</h1>
<img src="https://placehold.co/150" alt="My profile picture">
<p>I am a student passionate about web development. My hobbies include reading and playing games.</p>
<h2>Contact Me:</h2>
<a href="#">Facebook</a>

Thành quả mong muốn:

Bài tập - Bước 1: Tạo Container

Đầu tiên, bọc toàn bộ code HTML trong một thẻ <div> với class profile-container. Sau đó, dùng CSS để tạo một "tấm thẻ" và căn nó ra giữa màn hình. 

<!-- CSS -->
<style>
  body {
    background-color: #f3f4f6; /* Thêm màu nền cho đẹp */
    font-family: 'Inter', sans-serif;
  }
  .profile-container {
    max-width: 600px;
    margin: 40px auto; /* 40px trên dưới, auto trái phải để căn giữa */
    background-color: white;
    border-radius: 15px;
    box-shadow: 0 5px 15px rgba(0,0,0,0.05);
  }
</style>

<!-- HTML -->
<div class="profile-container">
    <h1>Hello, I'm John Doe</h1>
    <img src="https://placehold.co/150" alt="My profile picture">
    <p>I am a student passionate about web development...</p>
    <h2>Contact Me:</h2>
    <a href="#">Facebook</a>
</div>

Bài tập - Bước 2: Thêm Padding & Căn chỉnh nội dung

Nội dung đang dính sát vào mép thẻ. Hãy thêm padding cho container. Đồng thời, căn giữa toàn bộ nội dung bằng text-align: center.

<!-- Toàn bộ code cho Bước 2 -->
<style>
  body { 
    background-color: #f3f4f6; 
    font-family: 'Inter', sans-serif; 
  }
  .profile-container {
    max-width: 600px;
    margin: 40px auto;
    background-color: white;
    border-radius: 15px;
    box-shadow: 0 5px 15px rgba(0,0,0,0.05);
    /* -- THÊM MỚI -- */
    padding: 40px;
    text-align: center;
    box-sizing: border-box; /* Đảm bảo padding không làm tăng width */
  }
</style>

<!-- HTML -->
<div class="profile-container">
    <h1>Hello, I'm John Doe</h1>
    <img src="https://placehold.co/150" alt="My profile picture">
    <p>I am a student passionate about web development. My hobbies include reading and playing games.</p>
    <h2>Contact Me:</h2>
    <a href="#">Facebook</a>
</div>

Bài tập - Bước 3: Hoàn thiện Ảnh & Chữ

Bây giờ, hãy style riêng cho từng phần tử bên trong: biến ảnh đại diện thành hình tròn và tạo khoảng cách hợp lý cho các đoạn văn bản.

<!-- Toàn bộ code cho Bước 3 -->
<style>
  body { background-color:#f3f4f6; font-family:'Inter', sans-serif; }
  .profile-container {
    max-width: 600px; margin:40px auto;
    background:white; border-radius:15px;
    box-shadow:0 5px 15px rgba(0,0,0,0.05);
    padding:40px; text-align:center;
    box-sizing:border-box;
  }
  /* -- THÊM MỚI -- */
  .profile-picture {
    width: 150px; height:150px;
    border-radius: 50%; /* Đây là chìa khóa để tạo ảnh tròn */
    border: 5px solid #4f46e5;
    object-fit: cover; margin-bottom:20px;
  }
  .profile-name {
    font-size: 2rem; margin:0 0 10px 0;
  }
  .profile-bio {
    color: #4b5563; margin-bottom:25px;
  }
</style>

<!-- HTML với các class đã được thêm -->
<div class="profile-container">
  <img src="https://placehold.co/150" alt="Profile" class="profile-picture">
  <h1 class="profile-name">Hello, I'm John Doe</h1>
  <p class="profile-bio">I am a student passionate about web development. My hobbies include reading and playing games.</p>
  <h2>Contact Me:</h2>
  <a href="#">Facebook</a>
</div>
