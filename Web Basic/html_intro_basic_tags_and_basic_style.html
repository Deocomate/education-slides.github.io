Chào mừng bạn đến với buổi học đầu tiên về Lập trình Web!

1. Giới thiệu về HTML và CSS

HTML là gì?

HTML (HyperText Markup Language) là ngôn ngữ tiêu chuẩn để tạo ra các trang web.

  • Vai trò: Xây dựng cấu trúc, khung sườn cho nội dung trang web (tiêu đề, đoạn văn, hình ảnh, liên kết,...).
  • Ví dụ: Giống như bộ xương của cơ thể người, định hình và cấu trúc cơ bản.

CSS là gì?

CSS (Cascading Style Sheets) là ngôn ngữ dùng để mô tả cách trình bày các phần tử HTML trên màn hình.

  • Vai trò: Tạo kiểu dáng và bố cục cho trang web (màu sắc, font chữ, layout, khoảng cách,...).
  • Ví dụ: Giống như quần áo, màu da, kiểu tóc, làm cho "cơ thể" (trang web) trở nên đẹp mắt.

HTML cung cấp cấu trúc, CSS thêm vào phong cách, cùng nhau tạo nên một trang web hoàn chỉnh và trực quan.

2. Cú pháp và cấu trúc file HTML cơ bản

Cấu trúc file HTML cơ bản:

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Tiêu đề trang</title>
</head>
<body>
  Nội dung chính của trang web nằm ở đây
</body>
</html>
