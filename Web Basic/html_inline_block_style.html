Buổi 2: Cấu trúc với <div> & CSS Selectors

Làm chủ layout và lựa chọn phần tử

Hãy cùng xây dựng những layout web phức tạp hơn nhé!

1. Thẻ <div>: Người hùng thầm lặng của Layout

Thẻ <div> (division) là một "container" hoặc "hộp chứa" chung. Nó không có ý nghĩa về mặt ngữ nghĩa, vai trò chính của nó là để nhóm các phần tử khác lại với nhau nhằm mục đích tạo kiểu (CSS) hoặc xử lý (JavaScript).

Ứng dụng chính:

  • Tạo ra các khối layout lớn (header, main content, footer, sidebar).
  • Gom nhóm các nội dung liên quan để áp dụng chung một style.
  • Chia một trang web thành các phần riêng biệt, dễ quản lý.

Ví von:

Hãy tưởng tượng <div> như những chiếc hộp carton. Tự nó không có gì đặc biệt, nhưng bạn dùng chúng để sắp xếp đồ đạc (nội dung web) vào các nhóm ngăn nắp, giúp ngôi nhà (trang web) của bạn gọn gàng và có cấu trúc.

Ví dụ về bố cục trang web dùng <div>: 

<!-- Toàn bộ trang web được bọc trong một div container -->
<div class="container">
  <!-- Phần đầu trang -->
  <div class="header"> ...Nội dung header... </div>
  
  <!-- Phần thân trang -->
  <div class="main-content">
    <!-- Nội dung chính -->
    <div class="article"> ...Nội dung bài viết... </div>
    <!-- Thanh bên -->
    <div class="sidebar"> ...Nội dung sidebar... </div>
  </div>
  
  <!-- Phần chân trang -->
  <div class="footer"> ...Nội dung footer... </div>
</div>

2. Block vs. Inline: Hiểu cách các phần tử hiển thị

Mỗi phần tử HTML có một kiểu hiển thị mặc định. Hai kiểu phổ biến nhất là block và inline.

Phần tử Block

  • Luôn bắt đầu trên một dòng mới.
  • Chiếm toàn bộ chiều rộng của phần tử cha.
  • Ví dụ: <div>, <h1>-<h6>, <p>, <ul>, <li>.

Phần tử Inline

  • Không bắt đầu trên một dòng mới.
  • Chỉ chiếm chiều rộng cần thiết cho nội dung của nó.
  • Ví dụ: <span>, <a>, <img>, <strong>, <button>.

Ví dụ thực tế:

<!-- div và p là các phần tử block, chúng sẽ nằm trên các dòng riêng biệt -->
<div style="background-color: #f8d7da; padding: 10px; margin-bottom: 5px;">Đây là một div (block).</div>
<p style="background-color: #d1ecf1; padding: 10px;">Đây là một đoạn văn (block).</p>

<!-- span và a là các phần tử inline, chúng sẽ nằm trên cùng một dòng -->
<p style="background-color: #e2e3e5; padding: 10px; margin-top: 10px;">
  Đây là một
  <span style="background-color: #fff3cd; font-weight: bold; padding: 5px; border-radius: 3px;">span (inline)</span>
  và đây là một
  <a href="#" style="background-color: #d4edda; padding: 5px; border-radius: 3px;">liên kết (inline)</a>.
</p>

3. Viết CSS tập trung với thẻ <style>

Internal CSS: đặt các quy tắc CSS vào trong thẻ <style>, thường đặt trong phần <head>. Đây là cách làm tốt hơn Inline CSS ở Buổi 1.

Lợi ích:

  • Tách biệt nội dung (HTML) và trình bày (CSS).
  • Dễ dàng quản lý và chỉnh sửa tất cả style ở một nơi.
  • Có thể tái sử dụng một style cho nhiều phần tử.

Ví dụ: Chuyển từ Inline CSS sang Internal CSS 

<!DOCTYPE html>
<html>
<head>
  <title>Internal CSS</title>
  <style>
    h1 {
      color: navy;
      text-align: center;
    }
    p {
      font-family: Verdana;
      font-size: 16px;
    }
    button {
      background-color: green;
      color: white;
      padding: 10px;
      border: none;
      border-radius: 5px;
    }
  </style>
</head>
<body>
  <!-- HTML giờ đây trông gọn gàng hơn nhiều! -->
  <h1>Đây là Tiêu đề</h1>
  <p>Đây là một đoạn văn bản.</p>
  <button>Bấm vào tôi</button>
</body>
</html>

4. CSS Selector: Lựa chọn hàng loạt với Class

Khi bạn muốn áp dụng cùng một style cho nhiều phần tử khác nhau, hãy sử dụng thuộc tính class.

Cách hoạt động:

  1. Trong CSS, tạo một class với tên bắt đầu bằng dấu chấm (.).
  2. Trong HTML, thêm thuộc tính class="tên-class" vào các phần tử.
  3. Một phần tử có thể có nhiều class, cách nhau bởi dấu cách.

Ví dụ: Tạo các nút bấm và cảnh báo

<style>
  .btn {
    padding: 10px 15px;
    border: none;
    border-radius: 5px;
    font-weight: bold;
    cursor: pointer;
  }
  .btn-primary {
    background-color: #007bff;
    color: white;
  }
  .alert-danger {
    background-color: #f8d7da;
    color: #721c24;
    padding: 1rem;
    border: 1px solid #f5c6cb;
    border-radius: .25rem;
    margin-top: 15px;
  }
</style>
<!-- HTML -->
<button class="btn btn-primary">Lưu thay đổi</button>
<div class="alert-danger">
    <strong>Lỗi!</strong> Vui lòng kiểm tra lại thông tin.
</div>

5. CSS Selector: Lựa chọn duy nhất với ID

Khi bạn muốn áp dụng style cho một và chỉ một phần tử duy nhất trên trang, hãy sử dụng thuộc tính id.

Cách hoạt động:

  1. Trong HTML, thêm thuộc tính id="tên-id-duy-nhất". ID phải là duy nhất trên trang.
  2. Trong CSS, chọn phần tử bằng dấu thăng (#) theo sau là tên ID.

Ví dụ: Tạo kiểu cho Header

<style>
  #main-header {
    background-color: #343a40;
    color: white;
    padding: 20px;
    text-align: center;
    border-radius: 8px;
  }
</style>
<!-- HTML -->
<div id="main-header">
  <h1>Trang Web Của Tôi</h1>
</div>

6. Bài tập: Xây dựng Layout Blog Post

Áp dụng kiến thức đã học: <div>, class, và id để tạo một layout bài viết blog đơn giản. 

<!DOCTYPE html>
<html>
<head>
<style>
  body { background-color: #f4f4f4; color: #333; font-family: sans-serif; }
  #blog-container {
    width: 100%; /* Adjusted for output area */
    margin: auto;
    background-color: white;
    padding: 20px;
    box-shadow: 0 0 10px rgba(0,0,0,0.1);
    border-radius: 8px;
  }
  .post-header { border-bottom: 2px solid #eee; padding-bottom: 10px; margin-bottom: 20px; }
  .post-title { color: #0056b3; margin: 0; }
  .post-meta { font-size: 0.9em; color: #888; margin-top: 5px; }
  .post-content p { line-height: 1.6; }
  .author-box { background-color: #e9ecef; border: 1px solid #dee2e6; padding: 15px; margin-top: 25px; border-radius: 5px; }
</style>
</head>
<body>
  <div id="blog-container">
    <div class="post-header">
      <h1 class="post-title">Hành trình học Lập trình Web</h1>
      <p class="post-meta">Đăng bởi John Doe vào ngày 8 tháng 6, 2025</p>
    </div>
    <div class="post-content">
      <p>Đây là đoạn văn đầu tiên của bài viết. Chúng ta đang áp dụng các class để tạo kiểu.</p>
      <p>Thẻ div giúp chúng ta nhóm các phần tử lại với nhau.</p>
    </div>
    <div class="author-box">
      <h4>Về tác giả</h4>
      <p>John Doe là một nhà phát triển web đầy nhiệt huyết.</p>
    </div>
  </div>
</body>
</html>

Tổng kết Buổi 2

Thẻ <div> là công cụ để nhóm các phần tử và xây dựng layout.

Hiểu rõ Block (chiếm cả dòng) và Inline (nằm trên dòng) là chìa khóa để kiểm soát bố cục.

Sử dụng <style> giúp code HTML sạch hơn và CSS dễ quản lý hơn.

Class (.ten-class) dùng để áp dụng một style cho NHIỀU phần tử.

ID (#ten-id) dùng để định danh và tạo kiểu cho MỘT phần tử DUY NHẤT.
