Thêm sức sống với Hiệu ứng Hover & Transition

Hãy cùng xây dựng các phần tử đẹp mắt và có tính tương tác!

1. Ôn tập và Mục tiêu hôm nay

Trong các buổi trước, chúng ta đã học các kiến thức nền tảng:

  • Xây dựng cấu trúc nội dung với thẻ HTML.

  • Tạo bố cục với CSS Flexbox.

Mục tiêu hôm nay

2. Tương tác với `:hover`

Lớp giả (pseudo-class) :hover trong CSS được sử dụng để áp dụng các kiểu cho một phần tử khi con trỏ chuột của người dùng ở trên nó.

Đây là một trong những cách đơn giản nhất để tạo ra sự tương tác. 

<style>
.hover-link {
  color: #8b5cf6; /* Màu ban đầu */
  font-size: 1.5rem;
  font-weight: bold;
  text-decoration: none;
}
 
/* Kiểu này CHỈ áp dụng khi chuột di chuyển qua phần tử */
.hover-link:hover {
  color: #6d28d9; /* Màu thay đổi khi hover */
  text-decoration: underline;
}
</style>

3. Hiệu ứng mượt mà với `transition`

Sự thay đổi với :hover diễn ra tức thì và đột ngột. Thuộc tính transition cho phép bạn kiểm soát tốc độ thay đổi của thuộc tính, tạo ra các hoạt ảnh mượt mà.

Điều cần nhớ: Áp dụng thuộc tính `transition` cho bộ chọn cơ sở (ví dụ: `.my-button`), không phải bộ chọn `:hover`.

<style>
.box {
  width: 100px;
  height: 100px;
  background-color: #a78bfa;
  /* Định nghĩa transition:
   - all: áp dụng cho tất cả các thuộc tính thay đổi
   - 0.3s: thời gian kéo dài 0.3 giây
   - ease: hàm thời gian (bắt đầu chậm, nhanh, kết thúc chậm)
  */
  transition: all 0.3s ease;
}

.box:hover {
  background-color: #6d28d9;
  border-radius: 50%; /* Cũng sẽ được tạo hoạt ảnh */
  transform: scale(1.1); /* Thêm hiệu ứng thay đổi kích thước */
}
</style>
<div class="box"></div>

4. Thực hành: Xây dựng một Nút tương tác

Hãy xây dựng một thành phần UI phổ biến: nút bấm. Chúng ta sẽ áp dụng các kiểu cơ bản, sau đó thêm hiệu ứng hover và transition.

<style>
.my-button {
  /* Định dạng cơ bản */
  background-color: #8b5cf6;
  color: white;
  padding: 12px 24px;
  border: none;
  border-radius: 8px;
  font-size: 1rem;
  font-weight: 600;
  cursor: pointer;
  
  /* Thêm transition để có hiệu ứng mượt mà */
  transition: background-color 0.3s ease, transform 0.2s ease, box-shadow 0.2s ease;
}

.my-button:hover {
  /* Các kiểu khi chuột ở trên nút */
  background-color: #7c3aed;
  transform: translateY(-3px); /* Di chuyển nút lên một chút */
  box-shadow: 0 4px 10px rgba(0,0,0,0.15);
}
</style>

<button class="my-button">Nhấn vào tôi!</button>

Kết quả (Di chuột để xem hiệu ứng):

Các bước chính:

  1. Định nghĩa giao diện cơ bản (padding, color, border-radius).
  2. Thêm `transition` vào class cơ sở.
  3. Định nghĩa trạng thái `:hover` (thay đổi nền, thêm transform).

6. Thực hành: Xây dựng một Thành phần Thông báo (Alert)

Thông báo cung cấp các thông điệp phản hồi theo ngữ cảnh. Hãy tạo một thông báo "thành công" bằng Flexbox và thêm hiệu ứng hover tinh tế.

<style>

  display: flex; /* Dùng Flexbox để căn chỉnh icon và văn bản */
  align-items: center;
  padding: 1rem;
  border-radius: 8px;
  border-width: 1px;
  border-style: solid;
  max-width: 500px;

  transition: transform 0.2s ease-in-out, box-shadow 0.2s ease-in-out;

.alert:hover {

  margin-right: 0.75rem;

.alert-message {
  font-weight: 500;
}
</style>
