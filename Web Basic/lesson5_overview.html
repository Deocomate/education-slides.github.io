Bài Thi Cuối Khoá: Xây Dựng Website Profile

Hướng dẫn chi tiết để tạo trang giới thiệu bản thân từ A-Z

Bước 0: Giới Thiệu & Tiêu Chí Chấm Điểm

Mục tiêu của bài thi

Trong bài thi này, chúng ta sẽ vận dụng toàn bộ kiến thức đã học để xây dựng một sản phẩm thực tế: một trang web giới thiệu bản thân (Portfolio). Đây là "tấm danh thiếp" của bạn trong thế giới số, vì vậy hãy thật chăm chút cho nó nhé!

Tiêu chí chấm điểm

  • Hoàn Thiện Yêu Cầu Cơ Bản (7 điểm): Xây dựng trang web theo đúng các bước hướng dẫn, đảm bảo đầy đủ các thành phần và chức năng cơ bản.
  • Sáng Tạo (3 điểm+): Đây là phần để bạn thể hiện cá tính. Điểm cộng sẽ được trao cho những bạn thay đổi layout, màu sắc, font chữ, thêm hiệu ứng, bổ sung các thành phần mới, hoặc tự thiết kế một layout hoàn toàn mới.

Bước 1: Xây Dựng "Bộ Xương" - Cấu Trúc HTML

Khởi tạo dự án

Việc đầu tiên là tạo một thư mục cho dự án. Bên trong, hãy tạo 2 file:

  • index.html (chứa nội dung trang web)
  • style.css (chứa code CSS để trang trí)

Soạn cấu trúc HTML

Mở file index.html và sao chép đoạn code bên dưới. Đây là cấu trúc cơ bản cho trang profile của chúng ta, sử dụng các thẻ đã học như <div>, <img>, <h1>, và <a>.

Đừng quên thay thế các thông tin mẫu bằng thông tin cá nhân của bạn!

index.html 
<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Trang Cá Nhân Của [Tên Của Bạn]</title>
    <link rel="stylesheet" href="style.css">
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0-beta3/css/all.min.css">
</head>
<body>
    <div class="profile-card">
        <div class="info-section">
            <img class="profile-pic" src="https://placehold.co/150x150/4f46e5/FFFFFF?text=Avatar" alt="Ảnh đại diện">
            <h1 class="name">Nguyễn Văn A</h1>
            <p class="title">Lập Trình Viên Web</p>
            <p class="bio">Tôi là một người đam mê công nghệ và lập trình, luôn tìm tòi học hỏi những kiến thức mới.</p>
        </div>
        <div class="social-links">
            <a href="#"><i class="fab fa-facebook"></i></a>
            <a href="#"><i class="fab fa-github"></i></a>
            <a href="#"><i class="fab fa-linkedin"></i></a>
        </div>
    </div>
</body>
</html>

Bước 2: "Trang Điểm" Cho Card - Box Model & Layout

Tạo style cho trang

Mở file style.css. Chúng ta sẽ bắt đầu bằng việc định dạng cho <body> để có một nền đẹp và căn giữa nội dung bằng Flexbox.

Định hình thẻ Profile

Tiếp theo, ta sẽ style cho class .profile-card. Đây là lúc vận dụng kiến thức về Box Model.

style.css 
/* Reset cơ bản và font chữ */
body {
    font-family: 'Be Vietnam Pro', sans-serif;
    background-color: #f3f4f6;
    color: #333;
    display: flex;
    justify-content: center;
    align-items: center;
    min-height: 100vh;
    margin: 0;
    padding: 20px;
}

/* Style cho card chính */
.profile-card {
    background-color: white;
    max-width: 450px;
    width: 100%;
    padding: 40px;
    border-radius: 15px;
    box-shadow: 0 10px 30px rgba(0, 0, 0, 0.1);
    text-align: center;
    box-sizing: border-box;
}

Bước 3: Hoàn Thiện Giao Diện - Định Dạng Nội Dung

Làm đẹp cho ảnh và chữ

Bây giờ chúng ta sẽ đi sâu vào từng chi tiết bên trong card.

Style cho icon mạng xã hội

Cuối cùng, chúng ta sẽ định dạng cho các icon, chỉ cần tăng cỡ chữ và bỏ gạch chân mặc định của thẻ <a>.

style.css (viết tiếp) 

/* Style ảnh đại diện */
.profile-pic {
    width: 150px;
    height: 150px;
    border-radius: 50%;
    border: 5px solid #4f46e5;
    object-fit: cover;
    margin-bottom: 20px;
}

/* Style cho các phần text */
.name {
    margin: 0;
    font-size: 2.25rem;
    font-weight: 700;
}

.title {
    margin: 5px 0 15px 0;
    font-size: 1.1rem;
    color: #6b7280;
    font-weight: 500;
}

.bio {
    font-size: 1rem;
    line-height: 1.6;
    color: #4b5563;
    margin-bottom: 30px;
}

/* Style cho các liên kết MXH */
.social-links a {
    color: #4f46e5;
    font-size: 1.75rem;
    margin: 0 15px;
    text-decoration: none;
}

Bước 4: Thêm "Sự Sống" - Hiệu Ứng Hover & Transition

Tạo hiệu ứng mượt mà

Một trang web tĩnh thì khá nhàm chán. Chúng ta sẽ thêm "sự sống" bằng cách sử dụng transition và :hover.

style.css (viết tiếp) 

/* Áp dụng transition cho card ở trạng thái gốc */
.profile-card {
    transition: transform 0.3s ease, box-shadow 0.3s ease;
}

/* Style KHI di chuột vào card */
.profile-card:hover {
    transform: translateY(-10px);
    box-shadow: 0 20px 40px rgba(0, 0, 0, 0.15);
}

/* Áp dụng transition cho các icon */
.social-links a {
    transition: transform 0.2s ease, color 0.2s ease;
}

/* Style KHI di chuột vào icon */
.social-links a:hover {
    transform: scale(1.2);
    color: #3730a3;
}

Bước 5: Thử Thách Sáng Tạo - Thể Hiện Cá Tính Của Bạn

Thêm khu vực "Kỹ năng"

Đây là phần quan trọng nhất để bạn giành điểm cộng! Hãy thử thêm một khu vực hiển thị các kỹ năng của bạn.

index.html (thêm vào) 
<!-- Đặt đoạn code này dưới div.social-links -->
<div class="skills-section">
<h3 class="skills-title">Kỹ Năng Của Tôi</h3>
<div class="skills-list">
    <div class="skill-item">HTML5</div>
    <div class="skill-item">CSS3</div>
    <div class="skill-item">Flexbox</div>
    <div class="skill-item">JavaScript</div>
</div>
</div>
style.css (viết tiếp) 

.skills-section {
    margin-top: 30px;
    padding-top: 30px;
    border-top: 1px solid #e5e7eb;
}

.skills-title {
    margin-top: 0;
    margin-bottom: 20px;
    font-size: 1.5rem;
    color: #374151;
}

.skills-list {
    display: flex;
    flex-wrap: wrap;
    justify-content: center;
    gap: 10px;
}

.skill-item {
    background-color: #eef2ff;
    color: #4338ca;
    padding: 8px 15px;
    border-radius: 20px;
    font-weight: 600;
    font-size: 0.9rem;
    transition: transform 0.2s ease, background-color 0.2s ease;
}

.skill-item:hover {
    background-color: #c7d2fe;
    transform: scale(1.05);
}
