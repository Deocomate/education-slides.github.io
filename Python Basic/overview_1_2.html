Python For Beginners

Core Concepts Summary for Sessions 1 & 2

1. Variables and Data Types

A variable is like a labeled box used to store data. You can name a variable (following rules) and assign it a value.

Python has many data types, but here are the 4 most basic ones:

  • Integer (int): Whole numbers like -1, 0, 5, 100.
  • Float (float): Numbers with a decimal part, like 3.14, -0.5.
  • String (str): Text, enclosed in single quotes '' or double quotes "".
  • Boolean (bool): Has only two values: True or False.

Basic Functions:

We use built-in functions to interact with the user and process data. 

# 1. print(): Used to display data on the screen
print("Welcome to Python!")

# 2. input(): Used to get data from the user (always returns a string - str)
name = input("Please enter your name: ")
print("Hello,", name)

# 3. Type Casting: Converting data from one type to another
# Since input() always returns a string, we need to cast it for calculations
age_str = input("How old are you? ")
age_int = int(age_str) # Cast to an integer

height_str = input("How tall are you in meters? (e.g., 1.75) ")
height_float = float(height_str) # Cast to a float

# Print the casted results
print("So next year you will be", age_int + 1, "years old.")
print("Your height is:", height_float, "m.")

3. Conditional Statements: if / elif / else

Helps the program "make decisions", executing different blocks of code based on whether a condition is true or false.

  • if: If the condition is true, execute the code block inside.
  • elif: (else if) If the if condition is false, check this next condition. You can have multiple elifs.
  • else: If all of the above conditions are false, execute this code block.
score = float(input("Enter your score: "))

if score >= 8.0:
    print("You are an Excellent student!")
elif score >= 6.5:
    print("You are a Good student!")
elif score >= 5.0:
    print("You are an Average student.")
else:
    print("You need to try harder!")

4. Loops

a. The for Loop

Used to iterate over a sequence of elements (e.g., numbers in a range). Often used when the number of iterations is known beforehand.

The range(n) function generates a sequence of numbers from 0 to n-1.

# Print numbers from 0 to 4
print("For loop:")
for i in range(5):
    print("Current number is:", i)

# range(start, stop) -> iterates from start to stop-1
# Print numbers from 2 to 6
for num in range(2, 7):
    print(num)

b. The while Loop

Used to repeat a block of code AS LONG AS a condition is true. Often used when the number of iterations is not known beforehand.

Note: There must be a change within the loop that can make the condition False, otherwise it will cause an infinite loop!

# Countdown from 5 to 1
print("\nWhile loop:")
count = 5
while count > 0:
    print("Countdown:", count)
    count = count - 1 # Change the condition variable, very important!

print("Blast off!")

5. Homework Exercises

Complete the following exercises on your own to reinforce your knowledge. Don't be afraid to experiment and make mistakes; it's the best way to learn programming!

Exercise 1: Personal Information

Write a program that asks the user for their Name and Year of Birth. Then, calculate their age and print a welcome message containing both their name and age.

Exercise 2: Rectangle Area

Write a program that asks the user for the length and width of a rectangle. Calculate and print its area.

Exercise 3: Even or Odd Checker

Write a program that asks the user for an integer. Check if the number is even or odd and print the result.

Exercise 4: Temperature Converter

Write a program to convert Celsius to Fahrenheit. Ask the user for a temperature in Celsius. Formula: F = (C * 9/5) + 32.

Exercise 5: Find the Largest Number

Write a program that asks the user for 3 different numbers. Find and print the largest of the three.

Exercise 6: Simple Calculator

Write a program that asks the user for 2 numbers and an operator (+, -, *, /). Perform the corresponding calculation and print the result.

Exercise 7: Print a Sequence

Use a for loop to print all numbers from 1 to 10.

Exercise 8: Sum of Positive Numbers

Using a while loop, write a program that lets the user continuously enter numbers. The program should sum them up until the user enters 0, at which point it stops and prints the final sum.

Exercise 9: Factorial Calculator

Write a program that asks the user for a positive integer N. Calculate and print N! (N factorial). Example: 5! = 1 * 2 * 3 * 4 * 5 = 120.

Exercise 10: Multiplication Table

Write a program that asks the user for a number from 2 to 9. Print the multiplication table for that number (e.g., 7 x 1 = 7, 7 x 2 = 14, ..., 7 x 10 = 70).
