Python Programming Challenges

Practice your programming logic with hints and visual demonstrations.

Exercise 1: Odd or Even?

Easy

Objective: Write a function that takes an integer and checks if it's odd or even.

Exercise 2: Sum of a List

Easy

Objective: Given a list of integers, use a `for` loop to calculate the sum of all its elements.

Exercise 3: Find the Largest Number

Medium

Objective: Write a function that takes a list of numbers and returns the largest one without using the built-in `max()` function.

Hint

  • Assume the first element is the largest, then loop through the remaining elements to compare.

Exercise 4: Multiplication Table

Medium

Objective: Write a program that prints the multiplication tables from 2 to 9.

Hint

  • You will need two nested `for` loops.

Exercise 5: Drawing a Random Polygon

Medium

Objective: Write a function to draw a regular polygon with a random number of sides (from 3 to 8) and a random color.

Hint

  • The turning angle for each vertex of an N-sided regular polygon is `360 / N`.
  • Use the `random` library to pick the number of sides and the color.

Exercise 6: Drawing Nested Squares

Hard

Objective: Draw multiple squares, each rotated slightly from the previous one, to create a spiral effect.

Result Preview

loop 36 times:

  // Rotate for the next square

Exercise 7: Drawing a Circle from Squares

Hard

Objective: Write a function that draws a square, then call this function repeatedly, rotating slightly each time, to create a circular pattern.

Result Preview

Pseudocode / Logic

function draw_square(turtle, length):
  loop 4 times:
    turtle.forward(length)
    turtle.right(90)

// Main program
create a turtle
loop 72 times:
  call draw_square(my_turtle, 100)
  my_turtle.right(5)

Exercise 8: Turtle Race

Hard

Objective: Create a turtle race. Make multiple turtles, have them move forward by a random amount, and see which one reaches the finish line first.

Preview (Starting Line)

Pseudocode / Logic

// Turtle setup
create an empty list of turtles
for i in range(6):
  create a new_turtle, set its color, move to starting line
  add new_turtle to the list of turtles

// Start the race
while the race is not over:
  for each turtle in the list of turtles:
    check if turtle has crossed the finish line
    move_forward(a_random_distance)
