Buổi 1: Giới Thiệu Lập Trình Python

Biến, Kiểu dữ liệu, Toán tử và Nhập/Xuất Cơ Bản

Chào mừng các bạn đến với buổi học đầu tiên về Python!

1. Biến (Variables) trong Python

Biến là một tên được đặt cho một vị trí trong bộ nhớ máy tính, dùng để lưu trữ dữ liệu. Bạn có thể coi biến như một chiếc hộp có nhãn, bên trong chứa một giá trị nào đó.

Quy tắc đặt tên biến:

  • Tên biến có thể chứa chữ cái (a-z, A-Z), số (0-9) và dấu gạch dưới (_).
  • Tên biến không được bắt đầu bằng số.
  • Tên biến phân biệt chữ hoa, chữ thường (myVar khác myvar).
  • Không dùng các từ khóa của Python làm tên biến (ví dụ: if, else, for, while,...).
  • Nên đặt tên biến có ý nghĩa, dễ hiểu (ví dụ: ten_hoc_vien, diem_trung_binh). Quy ước phổ biến là dùng chữ thường và dấu gạch dưới (snake_case).

Ví dụ:

# Khai báo biến và gán giá trị
ten_hoc_vien = "Nguyễn Văn An"
tuoi = 20
diem_python = 8.5

# In giá trị của biến
print(ten_hoc_vien) # Output: Nguyễn Văn An
print(tuoi) # Output: 20

2. Kiểu Dữ Liệu (Data Types)

Kiểu dữ liệu xác định loại giá trị mà một biến có thể lưu trữ và các thao tác có thể thực hiện trên giá trị đó. Python có nhiều kiểu dữ liệu tích hợp sẵn.

Một số kiểu dữ liệu cơ bản thường dùng:

  • Số nguyên (Integer - int): Dùng để lưu trữ các số nguyên (không có phần thập phân). Ví dụ: 10, -5, 0.
  • Số thực (Floating-point - float): Dùng để lưu trữ các số có phần thập phân. Ví dụ: 3.14, -0.5, 2.0.
  • Chuỗi ký tự (String - str): Dùng để lưu trữ văn bản, được đặt trong dấu nháy đơn (') hoặc nháy kép ("). Ví dụ: "Xin chào", 'Python'.
  • Logic (Boolean - bool): Dùng để lưu trữ giá trị đúng/sai (True hoặc False). (Sẽ tìm hiểu kỹ hơn ở các buổi sau).

Python là ngôn ngữ kiểu động, nghĩa là bạn không cần khai báo tường minh kiểu dữ liệu của biến. Python sẽ tự động xác định kiểu dữ liệu dựa trên giá trị được gán.

x = 10 # x là kiểu int
y = 3.14 # y là kiểu float
ten = "Alice" # ten là kiểu str
la_sinh_vien = True # la_sinh_vien là kiểu bool

# Kiểm tra kiểu dữ liệu của biến bằng hàm type()
print(type(x)) # Output: <class 'int'>
print(type(y)) # Output: <class 'float'>
print(type(ten)) # Output: <class 'str'>

2.1. Kiểu Dữ Liệu Số Nguyên (int)

Kiểu int (integer) dùng để biểu diễn các số nguyên, không có phần thập phân.

Ví dụ:

so_luong_sach = 150
nam_hien_tai = 2024
nhiet_do_am = -10

print("Số lượng sách:", so_luong_sach) # Output: Số lượng sách: 150
print("Kiểu dữ liệu của nam_hien_tai:", type(nam_hien_tai)) # Output: Kiểu dữ liệu của nam_hien_tai: <class 'int'>

2.2. Kiểu Dữ Liệu Số Thực (float)

Kiểu float (floating-point number) dùng để biểu diễn các số thực, tức là các số có phần thập phân.

Ví dụ:

pi = 3.14159
diem_trung_binh = 7.85
gia_xang = 25000.0 # Cũng là float dù phần thập phân là .0

print("Giá trị Pi:", pi) # Output: Giá trị Pi: 3.14159
print("Kiểu dữ liệu của diem_trung_binh:", type(diem_trung_binh)) # Output: Kiểu dữ liệu của diem_trung_binh: <class 'float'>

2.3. Kiểu Dữ Liệu Chuỗi (str)

Kiểu str (string) dùng để biểu diễn văn bản hoặc một dãy các ký tự. Chuỗi được đặt trong dấu nháy đơn ('...') hoặc dấu nháy kép ("...").

Ví dụ:

loi_chao = "Xin chào các bạn!"
ten_sach = 'Lập trình Python cơ bản'
dia_chi = "123 Đường ABC, Quận XYZ"

print(loi_chao) # Output: Xin chào các bạn!
print("Kiểu dữ liệu của ten_sach:", type(ten_sach)) # Output: Kiểu dữ liệu của ten_sach: <class 'str'>

# Nối chuỗi (Concatenation)
ho = "Trần"
ten = "Văn"
dem = "Bình"
ho_ten = ho + " " + ten + " " + dem
print("Họ và tên:", ho_ten) # Output: Họ và tên: Trần Văn Bình

3. Toán Tử (Operators) trong Python

3.1. Toán Tử Số Học

Toán tử số học được sử dụng để thực hiện các phép toán toán học.

Toán tử Tên Ví dụ (a=10, b=3) Kết quả
+ Cộng a + b 13
- Trừ a - b 7
* Nhân a * b 30
/ Chia (kết quả là float) a / b 3.333...
// Chia lấy phần nguyên a // b 3
% Chia lấy phần dư (Modulo) a % b 1
** Lũy thừa a ** b 1000 (103)
a = 10
b = 3

print("a + b =", a + b) # Output: a + b = 13
print("a / b =", a / b) # Output: a / b = 3.3333333333333335
print("a // b =", a // b) # Output: a // b = 3
print("a % b =", a % b) # Output: a % b = 1

3.2. Phép Gán Giá Trị (Assignment Operator)

Toán tử gán chính là dấu bằng (=). Nó được sử dụng để gán giá trị cho biến.

x = 5 # Gán giá trị 5 cho biến x
y = x + 2 # Gán kết quả của (x + 2) cho biến y (y sẽ là 7)

print("Giá trị của x:", x) # Output: Giá trị của x: 5
print("Giá trị của y:", y) # Output: Giá trị của y: 7

Toán tử gán kết hợp (Compound Assignment Operators):

Python cung cấp các toán tử gán kết hợp để viết code ngắn gọn hơn.

Toán tử Tương đương với Ví dụ (cho x = 5)
x += 2 x = x + 2 x sẽ là 7
x -= 2 x = x - 2 x sẽ là 3
x *= 2 x = x * 2 x sẽ là 10
x /= 2 x = x / 2 x sẽ là 2.5
count = 0
count += 1 # Tương đương count = count + 1
print("Count:", count) # Output: Count: 1

total = 100
total -= 10 # Tương đương total = total - 10
print("Total:", total) # Output: Total: 90

4. Hàm print() - Xuất Dữ Liệu

Hàm print() được sử dụng để hiển thị (in) dữ liệu ra màn hình console.

Cách sử dụng:

  • In một chuỗi ký tự: print("Xin chào!")
  • In giá trị của biến: print(ten_bien)
  • In nhiều giá trị (các giá trị cách nhau bởi dấu phẩy, mặc định sẽ có khoảng trắng giữa chúng khi in): print("Tên:", ten, "Tuổi:", tuoi)
  • Định dạng chuỗi (f-string - cách hiện đại và dễ đọc): print(f"Bạn {ten} năm nay {tuoi} tuổi.")
ten = "Mai"
tuoi = 25

# In chuỗi cố định
print("Chào mừng đến với Python!")
# Output: Chào mừng đến với Python!

# In giá trị biến
print(ten)
# Output: Mai

# In nhiều giá trị
print("Tên của bạn là:", ten, ", bạn", tuoi, "tuổi.")
# Output: Tên của bạn là: Mai , bạn 25 tuổi.

# Sử dụng f-string (khuyến khích)
print(f"Xin chào {ten}, bạn năm nay {tuoi} tuổi.")
# Output: Xin chào Mai, bạn năm nay 25 tuổi.

so_a = 10
so_b = 20
print(f"Tổng của {so_a} và {so_b} là {so_a + so_b}")
# Output: Tổng của 10 và 20 là 30

5. Hàm input() - Nhập Dữ Liệu

Hàm input() được sử dụng để nhận dữ liệu từ người dùng qua bàn phím.

Đặc điểm quan trọng:

  • Hàm input() luôn trả về một chuỗi (str), ngay cả khi người dùng nhập số.
  • Bạn có thể truyền một chuỗi vào hàm input() để hiển thị lời nhắc cho người dùng. Ví dụ: input("Nhập tên của bạn: ")
  • Để sử dụng giá trị nhập vào dưới dạng số (int hoặc float), bạn cần ép kiểu (chuyển đổi kiểu) bằng hàm int() hoặc float().
# Nhập tên người dùng
ten_nguoi_dung = input("Vui lòng nhập tên của bạn: ")
print(f"Xin chào, {ten_nguoi_dung}!")

# Nhập tuổi và ép kiểu sang int
chuoi_tuoi = input("Bạn bao nhiêu tuổi? ")
tuoi_nguoi_dung = int(chuoi_tuoi) # Ép kiểu sang số nguyên
print(f"Vậy là bạn {tuoi_nguoi_dung} tuổi. Sang năm bạn sẽ {tuoi_nguoi_dung + 1} tuổi.")

# Nhập điểm trung bình và ép kiểu sang float
diem_tb_str = input("Nhập điểm trung bình của bạn: ")
diem_tb = float(diem_tb_str) # Ép kiểu sang số thực
print(f"Điểm trung bình của bạn là: {diem_tb}")

Lưu ý: Nếu người dùng nhập văn bản khi bạn cố gắng ép kiểu sang int() hoặc float(), chương trình sẽ báo lỗi (ValueError). Xử lý lỗi sẽ được học ở các buổi sau.

6. Ví dụ Tổng Hợp 1: Tính Tuổi

Yêu cầu: Viết chương trình nhập vào năm sinh của một người, sau đó tính và in ra tuổi của người đó (giả sử năm hiện tại là 2024).

nam_hien_tai = 2024

# Nhập năm sinh từ người dùng
chuoi_nam_sinh = input("Nhập năm sinh của bạn: ")

# Ép kiểu năm sinh sang số nguyên
nam_sinh = int(chuoi_nam_sinh)

# Tính tuổi
tuoi = nam_hien_tai - nam_sinh

# In kết quả
print(f"Bạn năm nay {tuoi} tuổi.")

Thử chạy chương trình:

Nếu bạn nhập năm sinh là 1995, kết quả sẽ là: Bạn năm nay 29 tuổi.

7. Ví dụ Tổng Hợp 2: Chào Hỏi Cá Nhân

Yêu cầu: Viết chương trình nhập vào tên và thành phố của một người, sau đó in ra một lời chào cá nhân hóa.

# Nhập tên từ người dùng
ten = input("Nhập tên của bạn: ")

# Nhập thành phố từ người dùng
thanh_pho = input("Bạn đến từ thành phố nào? ")

# In lời chào cá nhân hóa
print(f"Chào bạn {ten}! Rất vui được biết bạn đến từ {thanh_pho}.")

Thử chạy chương trình:

Nếu bạn nhập tên là Lan và thành phố là Hà Nội, kết quả sẽ là: Chào bạn Lan! Rất vui được biết bạn đến từ Hà Nội.

8. Bài Tập Thực Hành

Bài tập 1: Thông Tin Cá Nhân

Yêu cầu: Viết một chương trình Python thực hiện các bước sau:

  1. Yêu cầu người dùng nhập vào tên của họ.
  2. Yêu cầu người dùng nhập vào tuổi của họ.
  3. Yêu cầu người dùng nhập vào tên trường đại học (hoặc trường học/công ty) của họ.
  4. In ra một câu tổng hợp thông tin, ví dụ: "Chào bạn [Tên], bạn [Tuổi] tuổi và đang học/làm việc tại [Tên trường]."

Ví dụ output mong muốn:

# Input mẫu:
# Tên: An
# Tuổi: 21
# Trường: Đại học Bách Khoa

# Output mong muốn:
"Chào bạn An, bạn 21 tuổi và đang học/làm việc tại Đại học Bách Khoa."

8. Bài Tập Thực Hành (Tiếp theo)

Bài tập 2: Tính Diện Tích Hình Chữ Nhật

Yêu cầu: Viết một chương trình Python thực hiện các bước sau:

  1. Yêu cầu người dùng nhập vào chiều dài của hình chữ nhật (có thể là số thực).
  2. Yêu cầu người dùng nhập vào chiều rộng của hình chữ nhật (có thể là số thực).
  3. Tính diện tích của hình chữ nhật theo công thức: Diện tích = Chiều dài * Chiều rộng.
  4. In ra diện tích đã tính được, kèm theo thông tin rõ ràng.

Ví dụ output mong muốn:

# Input mẫu:
# Chiều dài: 10.5
# Chiều rộng: 5.2

# Output mong muốn:
"Diện tích hình chữ nhật với chiều dài 10.5 và chiều rộng 5.2 là: 54.6"

8. Bài Tập Thực Hành (Tiếp theo)

Bài tập 3: Chuyển Đổi Tiền Tệ (Đơn Giản)

Yêu cầu: Viết một chương trình Python thực hiện các bước sau:

  1. Yêu cầu người dùng nhập vào một số tiền bằng USD (Đô la Mỹ).
  2. Giả sử tỉ giá cố định là 1 USD = 23500 VND.
  3. Tính số tiền tương ứng bằng VND (Việt Nam Đồng).
  4. In ra số tiền đã quy đổi.

Ví dụ output mong muốn:

# Input mẫu:
# Số tiền USD: 50

# Output mong muốn:
"50 USD tương đương với 1175000 VND."

9. Tóm Tắt và Câu Hỏi

Trong buổi học hôm nay, chúng ta đã tìm hiểu về:

  • Biến: Cách đặt tên và lưu trữ dữ liệu.
  • Kiểu dữ liệu cơ bản: int (số nguyên), float (số thực), str (chuỗi).
  • Toán tử: Toán tử số học (+, -, *, /, //, %, **) và toán tử gán (=, +=, ...).
  • Hàm print(): Để xuất (in) dữ liệu ra màn hình.
  • Hàm input(): Để nhập dữ liệu từ người dùng và tầm quan trọng của việc ép kiểu.

Đây là những kiến thức nền tảng rất quan trọng để tiếp tục hành trình học lập trình Python của bạn!
