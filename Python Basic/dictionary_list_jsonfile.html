Buổi Học: Dictionary, List & JSON

Kết hợp cấu trúc dữ liệu và làm việc với file JSON để xây dựng ứng dụng

Hãy cùng xây dựng những ứng dụng thực tế đầu tiên!

1. Ôn Lại & Giới Thiệu

Ôn tập: Danh sách (List)

Ở Buổi Học trước, chúng ta đã học về List:

  • Là một tập hợp các phần tử có thứ tự.
  • Có thể thay đổi (thêm, sửa, xóa).
  • Truy cập bằng chỉ số (index) bắt đầu từ 0.
  • Ví dụ: sinh_vien = ["An", "Bình", "Cúc"]

Chủ đề mới: Từ điển (Dictionary)

Nếu List dùng số thứ tự (index) để quản lý dữ liệu, thì Dictionary dùng một thứ khác gọi là "key" (khóa).

Tưởng tượng một cuốn từ điển thật: bạn tra cứu một "từ" (key) để lấy "nghĩa" của nó (value).

Dictionary trong Python cũng vậy, nó lưu dữ liệu theo cặp key: value.

2. Tìm Hiểu về Dictionary

Dictionary là một cấu trúc dữ liệu dùng để lưu các giá trị theo cặp key: value.

Đặc điểm của Dictionary:

  • Không có thứ tự (Unordered): Các cặp key-value không được sắp xếp theo một thứ tự cụ thể (trong các phiên bản Python cũ). Từ Python 3.7+, dictionary giữ nguyên thứ tự được chèn vào.
  • Có thể thay đổi (Mutable): Bạn có thể thêm, sửa, xóa các cặp key-value.
  • Key là duy nhất (Unique Keys): Không thể có hai key trùng nhau trong một dictionary. Nếu bạn gán giá trị mới cho một key đã tồn tại, giá trị cũ sẽ bị ghi đè.
  • Key phải là bất biến (Immutable): Key thường là chuỗi (string), số (number), hoặc tuple. Bạn không thể dùng list làm key.

Cách khai báo Dictionary:

Sử dụng dấu ngoặc nhọn {} với các cặp key: value cách nhau bởi dấu phẩy.

# Dictionary rỗng
dict_rong = {}

# Một dictionary thông tin sinh viên
sinh_vien = {
    "ten": "Nguyễn Văn An",
    "ma_sv": "SV001",
    "tuoi": 20,
    "diem_trung_binh": 8.5,
    "da_tot_nghiep": False
}

print(sinh_vien)

2.1. Thao Tác Cơ Bản với Dictionary

1. Truy cập giá trị (Accessing Values):

Dùng key trong cặp ngoặc vuông [].

ten_sv = sinh_vien["ten"]
print(f"Tên sinh viên: {ten_sv}") # Output: Tên sinh viên: Nguyễn Văn An

2. Thêm hoặc cập nhật (Adding/Updating):

Gán giá trị cho một key. Nếu key chưa có, nó sẽ được thêm vào. Nếu đã có, giá trị sẽ được cập nhật.

# Cập nhật tuổi
sinh_vien["tuoi"] = 21
# Thêm quê quán
sinh_vien["que_quan"] = "Hà Nội"

print(sinh_vien)

3. Xóa một cặp key-value (Deleting):

Dùng từ khóa del.

del sinh_vien["da_tot_nghiep"]
print(sinh_vien)

3. Sức Mạnh Kết Hợp: List chứa Dictionary

Đây là một trong những cấu trúc phổ biến và hữu ích nhất trong Python!

Tưởng tượng bạn cần quản lý một danh sách lớp. Mỗi sinh viên có nhiều thông tin (tên, tuổi, điểm...). Thay vì tạo nhiều list riêng lẻ, chúng ta có thể tạo một list duy nhất, trong đó mỗi phần tử là một dictionary chứa thông tin của một sinh viên.

danh_sach_lop = [
    {
        "ten": "Nguyễn Văn An",
        "ma_sv": "SV001",
        "diem": 8.5
    },
    {
        "ten": "Trần Thị Bình",
        "ma_sv": "SV002",
        "diem": 9.0
    },
    {
        "ten": "Lê Văn Cúc",
        "ma_sv": "SV003",
        "diem": 7.8
    }
]

# In ra tên của sinh viên thứ hai (index 1)
print(danh_sach_lop[1]['ten'])  # Output: Trần Thị Bình

# In ra điểm của sinh viên thứ ba (index 2)
print(danh_sach_lop[2]['diem']) # Output: 7.8

Cấu trúc này giúp tổ chức dữ liệu một cách rõ ràng, logic và dễ quản lý.

3.1. Thao Tác Dữ Liệu: Thêm (Create)

Để thêm một sinh viên mới vào danh sách lớp, chúng ta chỉ cần tạo một dictionary mới cho sinh viên đó và dùng phương thức append() của list.

# danh_sach_lop từ slide trước...

# Tạo thông tin cho sinh viên mới
sinh_vien_moi = {
    "ten": "Phạm Thị Dung",
    "ma_sv": "SV004",
    "diem": 8.2
}

# Thêm sinh viên mới vào danh sách
danh_sach_lop.append(sinh_vien_moi)

# In ra toàn bộ danh sách để kiểm tra
for sv in danh_sach_lop:
    print(sv)

Bây giờ, danh_sach_lop đã có 4 sinh viên.

3.2. Thao Tác Dữ Liệu: Sửa (Update)

Để sửa thông tin của một sinh viên, chúng ta cần:

  1. Tìm đúng sinh viên đó trong list (thường là duyệt qua list và kiểm tra một giá trị định danh duy nhất như ma_sv).
  2. Sau khi tìm thấy dictionary của sinh viên đó, ta truy cập vào key cần sửa và gán giá trị mới.

Ví dụ: Cập nhật điểm cho sinh viên có mã 'SV001' thành 8.8

# danh_sach_lop từ slide trước...

for sinh_vien in danh_sach_lop:
    # Tìm sinh viên có mã SV001
    if sinh_vien['ma_sv'] == 'SV001':
        # Cập nhật điểm
        sinh_vien['diem'] = 8.8
        break # Dừng vòng lặp sau khi cập nhật xong để tối ưu

# In ra để kiểm tra
print(danh_sach_lop[0])
# Output: {'ten': 'Nguyễn Văn An', 'ma_sv': 'SV001', 'diem': 8.8}

3.3. Thao Tác Dữ Liệu: Xóa (Delete)

Để xóa một sinh viên, chúng ta cũng cần tìm ra sinh viên đó. Sau khi tìm thấy, ta có thể xóa dictionary tương ứng khỏi list.

Ví dụ: Xóa sinh viên có mã 'SV002'

Cách 1: Dùng remove() (Nếu bạn có chính xác đối tượng dictionary cần xóa)

# danh_sach_lop từ slide trước...
sinh_vien_can_xoa = None

for sinh_vien in danh_sach_lop:
    if sinh_vien['ma_sv'] == 'SV002':
        sinh_vien_can_xoa = sinh_vien
        break

if sinh_vien_can_xoa:
    danh_sach_lop.remove(sinh_vien_can_xoa)
    print("Đã xóa sinh viên SV002.")

print(f"Số lượng sinh viên còn lại: {len(danh_sach_lop)}")

Lưu ý: Việc xóa phần tử trong khi đang duyệt list có thể gây ra lỗi. An toàn hơn là tìm phần tử trước, sau đó mới xóa nó khỏi list.

4. Làm Việc với File JSON

JSON là gì?

JSON (JavaScript Object Notation) là một định dạng văn bản (text) rất nhẹ, được sử dụng rộng rãi để lưu trữ và trao đổi dữ liệu.

  • Con người có thể đọc và viết dễ dàng.
  • Máy tính có thể phân tích (parse) và tạo ra dễ dàng.
  • Nó là tiêu chuẩn mở, độc lập với ngôn ngữ lập trình.

JSON và Python Dictionary

Cú pháp của JSON rất giống với Dictionary của Python. Điều này làm cho việc chuyển đổi giữa chúng trở nên cực kỳ đơn giản.

Python Dictionary

{
    "ten": "Trà sữa",
    "gia": 50000,
    "con_hang": True
}

JSON Object

{
    "ten": "Trà sữa",
    "gia": 50000,
    "con_hang": true
}

Lưu ý nhỏ: True/False trong Python tương ứng với true/false (viết thường) trong JSON.

4.1. Đọc Dữ Liệu từ File JSON (Import)

Python có một thư viện tích hợp sẵn tên là json để làm việc này. Chúng ta sẽ dùng hàm json.load() để đọc dữ liệu từ một file.

Giả sử chúng ta có file data.json với nội dung:

[
    {
        "ten_mon": "Cà phê đen",
        "gia": 25000
    },
    {
        "ten_mon": "Bạc xỉu",
        "gia": 30000
    }
]

Code Python để đọc file này:

import json

# Mở file json ở chế độ đọc ('r')
with open('data.json', 'r', encoding='utf-8') as f:
    # Tải dữ liệu từ file và chuyển thành đối tượng Python
    thuc_don = json.load(f)

# Bây giờ, 'thuc_don' là một list các dictionary
print(thuc_don)
print(f"Tên món đầu tiên: {thuc_don[0]['ten_mon']}")

Lưu ý: encoding='utf-8' rất quan trọng để đọc file có chứa tiếng Việt.

4.2. Ghi Dữ Liệu ra File JSON (Export)

Để lưu một đối tượng Python (như list, dictionary) ra file JSON, ta dùng hàm json.dump().

Ví dụ: Lưu danh sách lớp học của chúng ta ra file danh_sach_lop.json 

import json

# Giả sử đây là danh sách lớp hiện tại
danh_sach_lop = [
    {'ten': 'Nguyễn Văn An', 'ma_sv': 'SV001', 'diem': 8.8},
    {'ten': 'Lê Văn Cúc', 'ma_sv': 'SV003', 'diem': 7.8}
]

# Mở file ở chế độ ghi ('w')
with open('danh_sach_lop.json', 'w', encoding='utf-8') as f:
    # Ghi đối tượng Python vào file JSON
    # indent=4: thụt lề 4 dấu cách để file dễ đọc hơn
    # ensure_ascii=False: để lưu tiếng Việt đúng định dạng
    json.dump(danh_sach_lop, f, indent=4, ensure_ascii=False)

print("Đã lưu danh sách lớp ra file danh_sach_lop.json thành công!")

Bây giờ bạn có thể mở file danh_sach_lop.json để xem kết quả.

5. Ứng Dụng: Quản Lý Sinh Viên

Hãy cùng nhau phác thảo một chương trình quản lý sinh viên hoàn chỉnh, sử dụng tất cả kiến thức vừa học.

Chức năng chính:

  • Hiển thị danh sách sinh viên.
  • Thêm một sinh viên mới.
  • Sửa thông tin một sinh viên (dựa vào mã SV).
  • Xóa một sinh viên (dựa vào mã SV).
  • Lưu danh sách hiện tại vào file JSON.
  • Tải danh sách từ file JSON khi chương trình bắt đầu.
  • Thoát chương trình.

Cấu trúc dữ liệu:

Một list chứa các dictionary, mỗi dictionary là một sinh viên.

[ {"ma_sv": "...", "ten": "...", "diem": ...}, ... ]

Luồng hoạt động:

Sử dụng vòng lặp while True để hiển thị menu và nhận lựa chọn của người dùng. Dùng if-elif-else để gọi các hàm xử lý tương ứng.

5. Ứng Dụng: Quản Lý Quán Cafe

Một ứng dụng khác là quản lý thực đơn (menu) của một quán cafe.

Chức năng chính:

  • Hiển thị toàn bộ thực đơn.
  • Thêm một món mới vào thực đơn (tên, giá, loại...).
  • Cập nhật giá của một món.
  • Xóa một món khỏi thực đơn.
  • Lưu và tải thực đơn từ file menu.json.

Cấu trúc dữ liệu:

Một list các dictionary, mỗi dictionary là một món đồ uống.

[ {"ten_mon": "...", "gia": ..., "loai": "..."}, ... ]

5. Ứng Dụng: Quản Lý Môn Học

Tương tự, ta có thể xây dựng chương trình quản lý các môn học trong một chương trình đào tạo.

Chức năng chính:

  • Hiển thị danh sách các môn học.
  • Thêm một môn học mới (mã môn, tên môn, số tín chỉ).
  • Sửa thông tin môn học (dựa vào mã môn).
  • Xóa một môn học.
  • Lưu và tải danh sách môn học từ file mon_hoc.json.

Cấu trúc dữ liệu:

Một list các dictionary, mỗi dictionary là một môn học.

[ {"ma_mon": "...", "ten_mon": "...", "tin_chi": ...}, ... ]

Tất cả các ứng dụng trên đều có chung một logic nền tảng: CRUD (Create, Read, Update, Delete) trên một list các dictionary và kết hợp với việc lưu/tải file JSON.

7. Tóm Tắt Buổi Học Học

Trong Buổi Học học hôm nay, chúng ta đã học được những kiến thức cốt lõi:

  • Dictionary (Từ điển): Cách lưu dữ liệu theo cặp key: value, các thao tác cơ bản như thêm, sửa, xóa, truy cập.
  • List chứa Dictionary: Cấu trúc dữ liệu cực kỳ mạnh mẽ để quản lý các đối tượng phức tạp (như sinh viên, sản phẩm...).
  • Thao tác CRUD trên List of Dictionaries: Cách thêm (Create), đọc (Read), cập nhật (Update) và xóa (Delete) dữ liệu trong cấu trúc này.
  • JSON: Hiểu được JSON là gì, tại sao nó quan trọng và sự tương đồng với Dictionary của Python.
  • Làm việc với file JSON: Sử dụng thư viện json để đọc (import) dữ liệu từ file vào chương trình và ghi (export) dữ liệu từ chương trình ra file.
  • Ứng dụng thực tế: Phác thảo cách xây dựng các chương trình quản lý dựa trên kiến thức đã học.

Nắm vững những khái niệm này là một bước tiến lớn để bạn có thể xây dựng các ứng dụng có ý nghĩa và xử lý dữ liệu thực tế.
