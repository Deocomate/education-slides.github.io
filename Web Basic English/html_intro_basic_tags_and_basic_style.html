1. Introduction to HTML and CSS

What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages.

  • Role: Builds the structure and framework for web page content (headings, paragraphs, images, links, etc.).
  • Example: Like the skeleton of the human body, defining its shape and basic structure.

What is CSS?

CSS (Cascading Style Sheets) is a language used to describe the presentation of HTML elements on screen.

  • Role: Creates the style and layout for the web page (colors, fonts, layout, spacing, etc.).
  • Example: Like clothes, skin color, hairstyle, making the "body" (web page) look attractive.

HTML provides the structure, CSS adds the style, together creating a complete and visual web page.

2. Basic HTML Syntax and File Structure

Basic HTML file structure:

<!DOCTYPE html> <html lang="en"> <!-- Changed lang to "en" --> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Page Title</title> <!-- Translated --> </head> <body> <!-- Main content of the web page goes here --> <!-- Translated --> Hello World! </body> </html>

Explanation:

  • <!DOCTYPE html>: Declares the document type, helping the browser understand it's an HTML5 document.
  • <html>: The root tag, wrapping all page content. The lang="en" attribute specifies the page language as English.
  • <head>: Contains meta-information about the page (not displayed directly), like title, encoding, CSS links, scripts.
  • <title>: Sets the page title, displayed on the browser tab.
  • <body>: Contains all visible content of the web page (text, images, links, etc.).

HTML Tag Syntax:

  • Most HTML tags have an opening tag (e.g., <p>) and a closing tag (e.g., </p>).
  • Content lies between the opening and closing tags.
  • Some tags are self-closing tags (void elements), having no content inside and not needing a separate closing tag (e.g., <img src="...">, <br>).
  • Tags can contain attributes, providing additional information to the tag. Attributes are written in the opening tag, in the form attribute_name="value".
<p class="description">This is a paragraph with a class attribute.</p> <!-- Translated example text --> <img src="image.jpg" alt="Image description"> <!-- Self-closing tag --> <!-- Translated comment -->

3. Basic Tags: Headings <h1> - <h6>

Heading tags are used to define heading levels in an HTML document. <h1> is the most important heading (highest level), and <h6> is the least important.

Using heading tags correctly improves page structure, is good for SEO (Search Engine Optimization), and accessibility.

Code Example:

<h1>This is Heading H1 (most important)</h1> <h2>This is Heading H2</h2> <h3>This is Heading H3</h3> <h4>This is Heading H4</h4> <h5>This is Heading H5</h5> <h6>This is Heading H6 (least important)</h6>

3. Basic Tags: Paragraph <p>

The <p> (paragraph) tag is used to define a paragraph of text.

Browsers automatically add some white space (margin) before and after each <p> element.

Code Example:

<p>This is the first paragraph. HTML is very interesting and is the foundation of the web.</p> <p>This is the second paragraph. Each <p> tag creates a separate block of text.</p>

Display Result:

This is the first paragraph. HTML is very interesting and is the foundation of the web.

This is the second paragraph. Each <p> tag creates a separate block of text.

3. Basic Tags: Link <a>

The <a> (anchor) tag is used to create hyperlinks to other web pages, files, email addresses, or locations within the same page.

Important attributes:

  • href: (Hypertext Reference) Specifies the URL (address) of the page the link goes to.
  • target="_blank": Opens the link in a new browser tab or window.
  • The content inside the <a>...</a> tags is the text or image that users will click on.

Code Example:

<a href="https://www.google.com">Visit Google</a> <br> <!-- Line break tag so links don't stick together --> <a href="https://www.bbc.com" target="_blank">Read news on BBC (new tab)</a> <br> <a href="mailto:info@example.com">Email Us</a>

Display Result:

Visit Google
Read news on BBC (new tab)
Email Us

Note: In the display example, actual links have been replaced with a JavaScript alert to avoid navigating away from this page.

3. Basic Tags: Image <img>

The <img> (image) tag is used to embed an image into a web page. It is a self-closing tag (no </img> closing tag).

Important attributes:

  • src: (Source) Specifies the path (URL) to the image file.
  • alt: (Alternative text) Provides a text description for the image. This text will be displayed if the image cannot be loaded and is very important for SEO and visually impaired users.
  • width, height: (Optional) Define the width and height of the image (in pixels or percentage). It's better to use CSS to control image sizes flexibly.

Code Example:

<img src="https://placehold.co/300x150/3498db/ffffff?text=Sample+Image+1" alt="Blue sample image"> <br> <img src="https://placehold.co/200x200/e74c3c/ffffff?text=Square+Image" alt="Red square image" width="100" height="100">

3. Basic Tags: Button <button>

The <button> tag is used to create a clickable button. Buttons are often used to trigger an action, usually via JavaScript (e.g., submitting a form, opening a dialog).

Basic attributes:

  • type: Defines the button type. Common values:
    • "button": A clickable button (default).
    • "submit": A button that submits form data.
    • "reset": A button that resets form fields to their initial values.
  • The content inside the <button>...</button> tags is the text or HTML that will be displayed on the button.

Code Example:

<button type="button" onclick="alert('You clicked the regular button!')">Click Me</button> <br><br> <button type="submit">Submit Data</button> <br><br> <button type="reset">Reset Form</button>

Note: The onclick attribute with alert() is added to illustrate the action when the button is clicked.

4. Basic CSS for Text Tags (style Attribute)

CSS (Cascading Style Sheets) is used to format the appearance of HTML. A simple way to add CSS is using inline CSS via the style attribute directly on an HTML tag.

Some basic CSS properties for text:

  • color: color_value;: Sets the text color. E.g., color: blue;, color: #FF0000; (hex code), color: rgb(0, 255, 0);.
  • font-size: size_value;: Sets the text size. E.g., font-size: 20px;, font-size: 1.5em;, font-size: 120%;.
  • font-family: font_name;: Sets the font type. E.g., font-family: Arial, sans-serif; (if Arial is not available, use the default sans-serif font).
  • text-align: alignment_value;: Aligns the text. E.g., text-align: center;, text-align: left;, text-align: right;.
  • font-weight: weight_value;: Sets the text boldness. E.g., font-weight: bold;, font-weight: normal;, font-weight: 700;.
  • background-color: color_value;: Sets the background color of the element.

Code Example:

<h1 style="color: green; text-align: center;">This heading is green and centered.</h1> <p style="font-size: 18px; color: #555555; font-family: 'Courier New', monospace; background-color: #f0f0f0; padding: 10px;"> This paragraph has a font size of 18px, dark gray color, Courier New font, light gray background, and 10px padding. </p>

Display Result:

This heading is green and centered.

This paragraph has a font size of 18px, dark gray color, Courier New font, light gray background, and 10px padding.

Note: Inline CSS is convenient for small changes, but for larger projects, it's better to use CSS in a <style> tag or a separate CSS file for easier management and reusability (to be covered in later sessions).

5. Example: Creating a Personal Introduction Page (HTML)

Let's apply the tags we've learned to create a simple personal introduction page using only HTML.

Requirements:

  • Use <h1> for your name.
  • Use <img> to display a profile picture (use a placeholder image).
  • Use <p> to write a few lines about yourself and your hobbies.
  • Use <h2> for a "Contact" section.
  • Use <a> to create links to your email or social media (simulated).

5. Example: Applying Basic CSS (style Attribute)

Now, let's add some inline CSS to the personal introduction page to make it look more appealing.

Requirements:

  • Change the font for the entire page.
  • Center the main heading (your name) and change its text color.
  • Round the profile picture and add a border.
  • Change the text color and font size for paragraphs.
  • Style the button (background color, text color, rounded corners).

HTML Code with Inline CSS:

I enjoy learning new things and challenging myself with interesting projects.</p> <p style="font-size: 1.1em; line-height: 1.7; color: #495057;">My hobbies include gaming, reading science fiction books, and traveling.</p> <h2 style="color: #28a745; margin-top: 30px;">Contact Me:</h2> <p style="font-size: 1.1em;"> You can connect with me via <a href="#" onclick="alert('This is a simulated Facebook link!')" style="color: #007bff; text-decoration: none; font-weight: bold;">Facebook</a> or email me at <a href="mailto:johndoe@example.com" style="color: #007bff; text-decoration: none; font-weight: bold;">johndoe@example.com</a>. </p> <button onclick="alert('Thanks for visiting my page!')" style="background-color: #28a745; color: white; padding: 12px 20px; border: none; border-radius: 5px; cursor: pointer; font-size: 1em; margin-top: 15px;">Click for a greeting!</button> </body> </html>

6. Practice Exercise

Your turn to practice! Apply what you've learned to create a simple web page on a topic you like.

Requirements:

  1. Choose a topic you like (e.g., a dish, movie, book, sports team, travel destination, etc.).
  2. Create a new HTML file (e.g., my_favorite_topic.html).
  3. Use the <h1> tag for the main title of the page (topic name).
  4. Write a few paragraphs describing the topic using <p> tags.
  5. Insert at least one relevant image using the <img> tag. Don't forget the alt attribute!
  6. Add an <a> tag linking to an external website with more information about the topic (open in a new tab).
  7. Create a <button> with a simple message (e.g., "Learn More").
  8. Use the style attribute to apply inline CSS:
    • Change the color and alignment for the <h1> title.
    • Change the font and font size for the <p> paragraphs.
    • Style the button (background color, text color).

Hint:

Be creative and experiment with different CSS properties! You can search for "CSS color picker" to choose color codes, or "CSS font list" for common fonts.

7. Summary and Q&A

In today's session, we covered the fundamental concepts of HTML and CSS:

  • What is HTML? HyperText Markup Language, used to build the structure (framework) of a web page.
  • What is CSS? Cascading Style Sheets, used to format and style (present) HTML elements.
  • Basic HTML file structure and the meaning of <!DOCTYPE>, <html>, <head>, <body>, <title> tags.
  • Basic HTML Tags:
    • Headings: <h1> to <h6>
    • Paragraphs: <p>
    • Links: <a> (with href, target attributes)
    • Images: <img> (with src, alt attributes)
    • Buttons: <button>
  • Inline CSS: Using the style attribute to apply CSS rules directly to HTML tags (e.g., color, font-size, text-align, background-color).

These are the first and crucial building blocks on your journey to mastering web programming. Keep practicing to solidify your knowledge!
